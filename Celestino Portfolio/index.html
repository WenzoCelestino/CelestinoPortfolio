My Role

Web Developer

Project: PNL System Design

Developed the PNL System Design (Profit & Loss Management System), a comprehensive web-based application similar to Excel but specifically customized for Appcase's business operations. The system automates complex pricing calculations, inventory management, and financial reporting with advanced features and modern UI/UX design.

Key Features Implemented:

  • Items Management: Complete CRUD operations for inventory items with automatic pricing calculations based on acquisition cost, contingency, cost of money, markup percentages, and VAT
  • Category-Based Pricing Rules: Implemented 9 different item categories (Imported Items, Local Materials, Controllers & Software, Services, Local Radios, etc.) each with customizable markup, partner commission, and customer commission percentages
  • Item Grouping System: Created functionality to group multiple items together for project-based pricing and management, with automatic category-based grouping
  • Global Parameters Management: Built system-wide configuration for contingency rates, cost of money, VAT percentage, and exchange rates with custom parameter support
  • Advanced Calculation Engine: Developed complex pricing formulas that calculate total price, VAT, and VAT-inclusive prices automatically based on category rules and global parameters
  • Image Upload & Management: Implemented image upload functionality for items with file management and retrieval system
  • Store Locations Management: Created location tracking system with map integration using Mapbox GL JS
  • Reports & Analytics: Built reporting dashboard showing total items, total value, group statistics, and average markup calculations
  • User Authentication: Implemented session-based authentication system with secure login functionality
  • Modern UI/UX: Designed responsive interface with advanced CSS animations, gradient effects, smooth transitions, and Excel-like table functionality
  • Data Persistence: Created JSON-based data storage system with automatic save/load functionality for categories, items, groups, and configurations
  • Search Functionality: Implemented real-time search across items with dynamic filtering capabilities
